## 🚀 Node.js 简介

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行时，使 JavaScript 能够在服务器端运行。

### Node.js 特点

- **事件驱动**：基于事件循环的非阻塞 I/O 模型
- **单线程**：主线程是单线程，通过事件循环处理并发
- **异步 I/O**：高效的异步操作，适合 I/O 密集型应用
- **丰富的生态**：npm 包管理器拥有庞大的包生态系统
- **跨平台**：支持 Windows、Linux、macOS 等平台
- **高性能**：V8 引擎提供出色的执行性能

### 与其他语言的对比

| 特性 | Node.js | Python | Java | Go |
|------|---------|--------|------|-----|
| 运行方式 | 解释执行（JIT） | 解释执行 | 编译到字节码（JVM） | 编译型 |
| 并发模型 | 事件循环 | 多线程/协程 | 多线程 | Goroutine |
| I/O 模型 | 异步非阻塞 | 同步/异步 | 同步/异步 | 异步 |
| 类型系统 | 动态类型 | 动态类型 | 静态类型 | 静态类型 |
| 包管理 | npm | pip | Maven/Gradle | go mod |
| 学习难度 | ⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐ |

### Node.js 适用场景

- **Web 服务器**：构建 RESTful API、微服务
- **实时应用**：聊天应用、实时协作工具
- **数据流处理**：日志处理、数据转换
- **CLI 工具**：命令行工具和脚本
- **IoT 应用**：物联网设备控制
- **代理服务器**：反向代理、API 网关

### Node.js 不适用场景

- **CPU 密集型任务**：图像处理、视频编码（建议使用 Worker Threads）
- **内存密集型应用**：大数据处理（单进程内存限制）
- **需要强类型检查**：大型企业应用（可考虑 TypeScript）

## 📝 快速开始

```bash
# 检查 Node.js 版本
node --version
# 输出：v22.x.x

# 检查 npm 版本
npm --version
# 输出：10.x.x

# 创建第一个 Node.js 项目
mkdir my-nodejs-app
cd my-nodejs-app
npm init -y

# 创建 index.js
echo "console.log('Hello, Node.js!');" > index.js

# 运行程序
node index.js
```
